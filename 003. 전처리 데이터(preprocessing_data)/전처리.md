**`master_grid` = 격자 기본 정보**
- 273,001개 격자 각각의 고정 정보
- 전신주 개수, 산림 여부, 고도, 경사도 등 18개 변수

## 7. 전처리 2단계 — 실효습도
> **파일**: `PREP/PREP_02_실효습도.ipynb`  
> **시각화**: `PREP/PREP_02_실효습도시계열.png`  
> **출력**: `derived/effective_humidity/` (64.7 MB)

### 왜 만들었나?

> **쉬운 설명**:  
> 오늘 습도가 50%라도, **어제·그제도 계속 건조했다면** 실제로는 훨씬 더 위험합니다.  
> 오늘 습도만 보면 이 사실을 모릅니다.

```
실효습도 = 오늘~4일 전 습도를 가중 합산
         = 0.3×오늘 + 0.21×어제 + 0.147×그저께 + 0.103×3일전 + 0.072×4일전
           (오늘이 가장 중요, 오래될수록 영향 줄어듦)
```

### 구현의 핵심 도전

월을 넘어갈 때 연속성 유지
```
2월 28일의 실효습도 계산 시 → 정상
3월  1일의 실효습도 계산 시 → 2월 마지막 4일 데이터 필요!

해결책: "버퍼" — 2월 마지막 4일을 임시 보관 후 3월 앞에 붙여서 계산
```

### 결과 (산림청 경보 기준)

| 월 | 평균 실효습도 | EH < 40% (주의보) | EH < 30% (경보) |
|----|------------|-------------------|----------------|
| **2월** | **46.1%** | **26.6%** | **10.4%** |
| 3월 | 51.5% | 9.0% | 0.7% |
| 4월 | 50.4% | 13.8% | 1.2% |
| 5월 | 58.4% | 2.8% | 0% |

> **2월이 가장 건조** = 실제 강원도 산불 위험 최고 시기와 일치

## 8. 전처리 3단계 — 강우효과 (RNE)
> **파일**: `PREP/PREP_03_강우효과RNE.ipynb`  
> **시각화**: `PREP/PREP_03_RNE시계열.png`  
> **출력**: `derived/rne/` (12.1 MB)

### 왜 만들었나?

> **쉬운 설명**:  
> 어제 폭우가 왔다면 숲이 아직 촉촉합니다. 오늘 날씨만 보면 이 사실을 모릅니다.  
> RNE는 **최근 3일 비가 산불 위험을 얼마나 낮추는지** 수치로 표현합니다.

```
최근 3일 비 없음 → RNE = 1.0 (위험 그대로)
최근 3일 폭우   → RNE = 0.1 (위험 90% 감소)
```

### 계산 방법

```
1단계: 강수량을 4개 등급으로 변환
  0~1mm  → 0등급
  1~5mm  → 1등급
  5~10mm → 2등급
  10mm+  → 3등급

2단계: 3일치 등급 합산 → 표에서 RNE 조회
  등급합 0~1 → RNE 1.0
  등급합 2   → RNE 0.5
  등급합 6+  → RNE 0.1
```

### ★ 놀라운 발견

```
2월 RNE = 1.0이 100%  ←  전체 28일 동안 비가 1mm도 안 왔습니다
5월은 28.7%만 RNE = 1.0  ←  봄비가 오기 시작
```

이 결과 자체가 강원도 2~3월이 산불 위험 최고인 이유를 데이터로 증명합니다.

## 9. 전처리 4단계 — 일 가중치 (w_d)

앞의 실효습도·RNE는 3,276만 행이었지만,

3/11 ~ 3/20  : w_d = 0.95
3/21 ~ 4/10  : w_d = 1.00  ← 연중 최고 위험기 (21일)
4/11 ~ 4/20  : w_d = 0.95
4/21 ~ 4/30  : w_d = 0.90  ↓ 내려오기 시작
5/01 ~ 5/31  : w_d = 0.85

## 10. 전처리 5단계 — DWI (기상위험지수)
> **파일**: `PREP/PREP_05_DWI기상위험지수.ipynb`  
> **시각화**: `PREP/PREP_05_DWI등급분포.png`, `PREP_05_DWI시계열.png`  
> **출력**: `derived/dwi/` (201 MB)

### 드디어 앞의 모든 것을 합칩니다

```
사용한 재료          만든 곳
──────────────────────────────────────────
기온 (ta_mean)       → weather_scaled/
습도 (hm_mean)       → weather_scaled/
풍속 (wind_ws_mean)  → weather_scaled/
실효습도 (EH)        → derived/effective_humidity/
강우효과 (RNE)       → derived/rne/
```

### 계산 2단계

```
1단계: PreDWI = sigmoid(2.706 + 0.088×기온 - 0.055×습도
                                - 0.023×실효습도 - 0.104×풍속)
       결과: 0~1 사이 연속값
       (건조·고온일수록 1에 가까움)

2단계: PreDWI → 1~10 등급으로 재분류 (산림청 기준)
       DWI = 등급 × RNE
       결과: 0.1~10 (높을수록 위험)
```

### 최종 결과

| 월 | 평균 DWI | 고위험(DWI≥8) 격자 | 해석 |
|----|---------|------------------|------|
| 2월 | 2.66 | 4.9% | 기온 낮아 PreDWI 낮음 |
| 3월 | 3.00 | 6.5% | 건조 지속 |
| **4월** | **4.11** | **13.7%** | ★ 기온↑ + 건조 = 최고 위험 |
| 5월 | 2.81 | 4.2% | 봄비 효과 |

**4월 고위험 13.7% = w_d 최고점(1.0) 기간과 정확히 일치**

## 12. 다음 단계 (기상 파트 완료 후 남은 것)

```
현재까지 완료 (기상 파트)
  DWI = PreDWI_등급 × RNE  ✅

P-FFDRI 완성을 위해 추가 필요
  FMI (임상위험지수)    = 침엽수림 10, 혼효림 3, 활엽수림 2
  TMI (지형위험지수)    = 경사도 + 고도 + 사면방향 + 양간지풍
  PEI (전력설비노출지수) = 전신주 밀집도 + (화재이력 KDE — 고민 중)

최종 공식
  P-FFDRI = (0.55×DWI + 0.10×FMI + 0.15×TMI + 0.20×PEI) × w_d
```
