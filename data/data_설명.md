## 2. 데이터 출처

원본 데이터는 **기상청 API 허브**에서 수집했습니다.

- 기상청 API 허브: https://apihub.kma.go.kr/

수집한 데이터 항목은 다음과 같습니다.

| 파일 | API 허브 경로 |
|---|---|
| AWS지점정보 | 지상관측 → 지상관측 지점정보 → 1. 지상관측 지점정보 조회 → 1.1.2 AWS |
| 기온 | 지상관측 → 기후통계 → 1. 기온 기후통계데이터 조회 → 1.1.1 기온 기후통계 데이터 일자료 조회(전체지점) |
| 바람 | 지상관측 → 기후통계 → 5. 바람 기후통계데이터 조회 → 5.1.1 바람 기후통계 데이터 일자료 조회(전체지점) |
| 습도 | 지상관측 → 기후통계 → 10. 습도 기후통계데이터 조회 → 10.1.1 습도 기후통계 데이터 일자료 조회(전체지점) |
| 강수량 | 지상관측 → 기후통계 → 14. 강수량 기후통계데이터 조회 → 14.1.1 강수량 기후통계 데이터 일자료 조회(전체지점) |

## 3. 데이터 범위

- 기간: **2019년 11월 01일 ~ 2025년 12월 31일**
- 지역: **강원특별자치도**
- 단위: 일 단위 관측 데이터

기온, 바람, 습도, 강수량 데이터는 전체 지점 자료에서 **강원특별자치도 지역 지점만 필터링**한 데이터입니다.
