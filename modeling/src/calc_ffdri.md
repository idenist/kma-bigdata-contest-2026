# `calc_ffdri.py` 설명 문서

## 1. 목적

`calc_ffdri.py`는 강원도 100m 격자 단위 기상 데이터와 격자 마스터 데이터를 이용하여 산림청 산불위험지수인 `FFDRI`를 계산하고, 월별 Parquet 파일로 저장하는 스크립트입니다.

이 스크립트는 다음 지표를 계산합니다.

```text
DWI  : 기상위험지수
FMI  : 임상위험지수
TMI  : 지형위험지수
FFDRI: 최종 산불위험지수
```

최종 산출식은 다음과 같습니다.

```text
FFDRI = {7 × DWI + 1.5 × FMI + 1.5 × TMI} × 일가중치
```

본 스크립트에서는 전신주 노출지수인 `PEI`를 포함하지 않습니다.
즉, 기존 산림청 산불위험지수 구조인 `FFDRI`만 계산합니다.

## 2. 입력 데이터

### 2.1 날씨 데이터

```text
processed/grid_date_master/
```

날씨 데이터는 월별 파티션 형태로 저장되어 있습니다.

```text
processed/grid_date_master/
├─ month=2020-02/
│  ├─ *.parquet
├─ month=2020-03/
│  ├─ *.parquet
...
├─ month=2025-05/
│  ├─ *.parquet
```

각 월 폴더 안에는 여러 개의 Parquet 파일이 들어 있습니다.

주요 컬럼은 다음과 같습니다.

| 컬럼명                | 설명          |
| ------------------ | ----------- |
| `grid_id`          | 100m 격자 ID  |
| `date`             | 날짜          |
| `kma_nx`           | 기상청 격자 X    |
| `kma_ny`           | 기상청 격자 Y    |
| `ta_mean`          | 일평균 기온      |
| `ta_max`           | 일최고 기온      |
| `hm_mean`          | 일평균 상대습도    |
| `hm_min`           | 일최저 상대습도    |
| `td_mean`          | 일평균 이슬점온도   |
| `td_min`           | 일최저 이슬점온도   |
| `wind_ws_mean`     | 일평균 풍속      |
| `wind_ws_max`      | 일최대 풍속      |
| `wind_uu_mean`     | 일평균 U 성분 풍속 |
| `wind_vv_mean`     | 일평균 V 성분 풍속 |
| `wind_wd_sin_mean` | 풍향 sin 평균   |
| `wind_wd_cos_mean` | 풍향 cos 평균   |
| `rn_day_mean`      | 일평균 강수량     |
| `rn_day_max`       | 일최대 강수량     |

기상 값은 원본 저장 시 `data_scale = 10`이 적용되어 있으므로, 실제 값으로 사용하기 위해 `/10` 처리를 수행합니다.

단, 다음 컬럼은 `/10` 처리하지 않습니다.

```text
grid_id
date
month
kma_nx
kma_ny
wind_wd_sin_mean
wind_wd_cos_mean
```

### 2.2 격자 마스터 데이터

```text
processed/master_grid.parquet
```

`master_grid.parquet`는 강원도 100m 격자의 정적 정보를 담고 있습니다.

주요 컬럼은 다음과 같습니다.

| 컬럼명                | 설명         |
| ------------------ | ---------- |
| `grid_id`          | 100m 격자 ID |
| `is_forest`        | 산림 여부      |
| `forest_type_code` | 임상 코드      |
| `elevation`        | 고도         |
| `slope`            | 경사도        |
| `aspect_sin`       | 사면방향 sin   |
| `aspect_cos`       | 사면방향 cos   |

본 스크립트에서는 `master_grid.parquet`에 존재하는 격자만 사용합니다.
즉, 날씨 데이터에 다른 지역의 격자가 포함되어 있더라도, `master_grid`와 `INNER JOIN`하여 강원도 분석 대상 격자만 남깁니다.

## 3. 출력 데이터

계산 결과는 다음 경로에 저장됩니다.

```text
processed/ffdri/
```

출력 구조는 월별 폴더로 저장됩니다.

권장 구조는 다음과 같습니다.

```text
processed/ffdri/
├─ 2020-02/
│  └─ part-0.parquet
├─ 2020-03/
│  └─ part-0.parquet
...
├─ 2025-05/
│  └─ part-0.parquet
```

주의할 점은, 출력 폴더명을 `month=2025-03`처럼 저장하면 `month` 파티션 컬럼과 파일 내부 `month` 컬럼이 충돌할 수 있다는 점입니다.

따라서 출력 폴더는 다음처럼 저장하는 것을 권장합니다.

```text
processed/ffdri/2025-03/
```

이렇게 하면 `month` 컬럼 타입 충돌 문제를 피할 수 있습니다.

## 4. 전체 처리 흐름

`calc_ffdri.py`의 처리 흐름은 다음과 같습니다.

```text
1. 처리 가능한 월 목록 탐색
2. 월별 날씨 데이터 로드
3. data_scale = 10 적용
4. 실효습도 계산
5. 강우효과 RNE 계산
6. PreDWI 계산
7. PreDWI를 1~10등급으로 재분류
8. DWI 계산
9. master_grid에서 FMI 계산
10. master_grid에서 TMI 계산
11. 일가중치 계산
12. FFDRI 계산
13. 월별 Parquet 저장
```

## 5. 세부 계산 방식

## 5.1 data_scale 적용

기상 값은 원본에서 10배 스케일로 저장되어 있으므로 다음 컬럼에 대해 `/10`을 적용합니다.

```text
ta_mean
ta_max
hm_mean
hm_min
td_mean
td_min
wind_ws_mean
wind_ws_max
wind_uu_mean
wind_vv_mean
rn_day_mean
rn_day_max
```

예시는 다음과 같습니다.

```text
125  → 12.5℃
650  → 65.0%
35   → 3.5m/s
```

---

## 5.2 실효습도 계산

실효습도는 오늘의 습도뿐 아니라 이전 며칠간의 습도 영향을 반영하는 파생변수입니다.

계산식은 다음과 같습니다.

```text
실효습도 =
0.300 × 오늘 습도
+ 0.210 × 1일 전 습도
+ 0.147 × 2일 전 습도
+ 0.103 × 3일 전 습도
+ 0.072 × 4일 전 습도
```

월 초의 실효습도를 계산하려면 이전 달 마지막 4일 데이터가 필요합니다.
따라서 스크립트는 특정 월을 계산할 때, 해당 월 시작일보다 4일 전부터 데이터를 함께 읽습니다.

예를 들어 `2025-03`을 계산할 경우 다음 기간을 읽습니다.

```text
읽는 기간: 2025-02-25 ~ 2025-03-31
저장 기간: 2025-03-01 ~ 2025-03-31
```

---

## 5.3 강우효과 RNE 계산

먼저 일강수량을 등급으로 변환합니다.

|           일강수량 | 강수 등급 |
| -------------: | ----: |
|  0mm 이상 1mm 미만 |     0 |
|  1mm 이상 5mm 미만 |     1 |
| 5mm 이상 10mm 미만 |     2 |
|        10mm 이상 |     3 |

그 다음 최근 3일의 강수 등급을 합산합니다.

```text
RNE_Temp = 오늘 강수등급 + 1일 전 강수등급 + 2일 전 강수등급
```

`RNE_Temp`를 이용하여 강우효과 `RNE`를 계산합니다.

|  RNE_Temp | RNE |
| --------: | --: |
| 0 이상 2 미만 | 1.0 |
| 2 이상 3 미만 | 0.5 |
| 3 이상 4 미만 | 0.4 |
| 4 이상 5 미만 | 0.3 |
| 5 이상 6 미만 | 0.2 |
|      6 이상 | 0.1 |

`RNE`는 비가 산불 위험을 얼마나 낮추는지를 나타내는 계수입니다.
값이 작을수록 최근 강수로 인해 산불 위험이 낮아졌다는 의미입니다.

---

## 5.4 PreDWI 계산

현재 분석 대상 기간은 2월~5월이므로 봄철 공식을 사용합니다.

```text
PreDWI = 1 / (1 + exp(-(2.706
                      + 0.088 × 평균기온
                      - 0.055 × 상대습도
                      - 0.023 × 실효습도
                      - 0.104 × 평균풍속)))
```

사용 컬럼은 다음과 같습니다.

| 항목   | 사용 컬럼                |
| ---- | -------------------- |
| 평균기온 | `ta_mean`            |
| 상대습도 | `hm_mean`            |
| 실효습도 | `effective_humidity` |
| 평균풍속 | `wind_ws_mean`       |

---

## 5.5 PreDWI 재분류

계산된 `PreDWI`는 0~1 사이의 연속값입니다.
이를 산림청 기준에 따라 1~10등급으로 재분류합니다.

|       PreDWI 구간 | 등급 |
| --------------: | -: |
| 0.0000 ~ 0.1183 |  1 |
| 0.1184 ~ 0.1878 |  2 |
| 0.1879 ~ 0.2571 |  3 |
| 0.2572 ~ 0.3320 |  4 |
| 0.3321 ~ 0.4089 |  5 |
| 0.4090 ~ 0.4932 |  6 |
| 0.4933 ~ 0.5861 |  7 |
| 0.5862 ~ 0.6862 |  8 |
| 0.6863 ~ 0.7820 |  9 |
| 0.7821 ~ 1.0000 | 10 |

---

## 5.6 DWI 계산

DWI는 재분류된 PreDWI 등급에 강우효과 RNE를 곱하여 계산합니다.

```text
DWI = PreDWI 등급 × RNE
```

예를 들어, PreDWI 등급이 8이고 RNE가 0.5이면 다음과 같습니다.

```text
DWI = 8 × 0.5 = 4
```

---

## 5.7 FMI 계산

FMI는 임상위험지수입니다.

`master_grid.parquet`의 `forest_type_code`를 이용하여 계산합니다.

| forest_type_code | 임상 유형  | FMI |
| ---------------: | ------ | --: |
|                1 | 침엽수림   |  10 |
|                2 | 활엽수림   |   2 |
|                3 | 혼효림    |   3 |
|                4 | 죽림     |   1 |
|              그 외 | 비산림/기타 |   0 |

산불위험지수 계산에서는 침엽수림이 가장 높은 위험도를 갖습니다.

---

## 5.8 TMI 계산

TMI는 지형위험지수입니다.

본 스크립트에서는 고도지수와 방위지수를 합산하여 계산합니다.

```text
TMI = 고도지수 + 방위지수
```

### 고도지수

|              고도 | 고도지수 |
| --------------: | ---: |
|         876m 이상 |    1 |
| 628m 이상 876m 미만 |    2 |
| 380m 이상 628m 미만 |    3 |
| 132m 이상 380m 미만 |    4 |
|         132m 미만 |    5 |

고도가 낮을수록 높은 점수를 부여합니다.

### 방위지수

`aspect_sin`, `aspect_cos`를 이용해 방위각을 계산합니다.

```text
aspect_deg = (degrees(atan2(aspect_sin, aspect_cos)) + 360) % 360
```

방위각은 다음 기준으로 8방위로 나눕니다.

| 방위 |                       각도 범위 | 방위지수 |
| -- | --------------------------: | ---: |
| N  | 337.5° ~ 360° 또는 0° ~ 22.5° |  2.5 |
| NE |               22.5° ~ 67.5° |  4.5 |
| E  |              67.5° ~ 112.5° |  1.5 |
| SE |             112.5° ~ 157.5° |  4.0 |
| S  |             157.5° ~ 202.5° |  4.0 |
| SW |             202.5° ~ 247.5° |  5.0 |
| W  |             247.5° ~ 292.5° |  2.5 |
| NW |             292.5° ~ 337.5° |  4.5 |

## 5.10 FFDRI 계산

최종 FFDRI는 다음 공식으로 계산합니다.

```text
FFDRI = {7 × DWI + 1.5 × FMI + 1.5 × TMI} × 일가중치
```

각 항목의 의미는 다음과 같습니다.

| 지표   | 의미          |
| ---- | ----------- |
| DWI  | 기상위험지수      |
| FMI  | 임상위험지수      |
| TMI  | 지형위험지수      |
| 일가중치 | 계절별 산불위험 보정 |

## 9. 주의사항

### 9.1 출력 폴더명 주의

출력 폴더를 다음처럼 만들면 `month` 컬럼 충돌이 발생할 수 있습니다.

```text
processed/ffdri/month=2025-03/
```

이 경우 pandas 또는 pyarrow가 폴더명에서 `month` 파티션 컬럼을 자동 생성하면서, 파일 내부 `month` 컬럼과 충돌할 수 있습니다.

따라서 다음 구조를 권장합니다.

```text
processed/ffdri/2025-03/
```

---

### 9.2 TMI 정의 확인 필요

본 스크립트에서는 다음 방식으로 TMI를 계산합니다.

```text
TMI = 고도지수 + 방위지수
```

다만 팀에서 사용하는 TMI 정의가 다르다면 `terrain_calc` 부분을 수정해야 합니다.

---

### 9.3 강원도 필터링

본 스크립트는 `master_grid.parquet`와 `INNER JOIN`하여, `master_grid`에 존재하는 격자만 결과에 포함합니다.

따라서 `master_grid`가 강원도 격자만 포함하고 있다면, 출력 결과도 강원도 지역으로 한정됩니다.

---

### 9.4 월 초 계산

실효습도 계산에는 이전 4일의 습도 데이터가 필요합니다.
따라서 특정 월을 계산할 때 해당 월 시작일보다 4일 전부터 데이터를 함께 읽습니다.

단, 전체 데이터의 첫 월인 경우 이전 월 데이터가 없을 수 있습니다.
이 경우 부족한 과거 습도는 현재일 습도로 대체됩니다.

---

## 10. 향후 확장 방향

현재 스크립트는 기본 `FFDRI`만 계산합니다.

향후 전신주 위험도를 반영하려면 다음과 같은 지표를 추가할 수 있습니다.

```text
PEI: 전력설비노출지수
P-FFDRI: 전신주 노출을 반영한 확장 산불위험지수
```

예상 확장식은 다음과 같습니다.

```text
P-FFDRI = (0.55 × DWI + 0.10 × FMI + 0.15 × TMI + 0.20 × PEI) × 일가중치
```

다만 현재 버전에서는 PEI와 P-FFDRI를 계산하지 않습니다.
